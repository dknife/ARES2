블록 코딩 인터페이스

Google Blockly 기반의 시각적 프로그래밍 환경으로, 코딩 경험이 없는 초등학생도 쉽게 로버를 제어할 수 있습니다.

📡

무선 블루투스 제어

Web Bluetooth를 통해 브라우저에서 직접 로버와 통신합니다. 별도 앱 설치 없이 웹만으로 제어할 수 있습니다.

🤖

AI 코딩 도우미

로컬 AI 모델이 블록 코딩에 대한 질문에 답변합니다. 아이들이 로봇 제어 방법을 쉽게 이해할 수 있도록 도와줍니다.

Architecture

시스템 구조

Web UI (Blockly) → Web Bluetooth (BLE) → Pico (UART 9600) → Hardware Modules
센서 데이터 → CommandProcessor → BLE 응답 (20B) → Web UI 표시
Raspberry Pi Pico MicroPython Web Bluetooth Google Blockly BLE / UART HM-10 / BT05 SSD1306 OLED HC-SR04 EXAONE 3.5 LLM
Components

주요 구성 요소

서보 모터 (바퀴)

360도 연속 회전 서보 2개로 전진, 후진, 좌회전, 우회전을 수행합니다.

DC 모터

메인 구동 모터로 강한 힘을 제공합니다. PWM 속도 제어와 방향 전환을 지원합니다.

초음파 센서 (HC-SR04)

전방 물체까지의 거리를 cm 단위로 측정합니다. 자동 정지 기능과 연동됩니다.

자기장 센서

자석의 존재를 감지하여 탐사 미션에 활용됩니다.

LED 배열 (5개)

5개의 LED를 개별 또는 전체 제어합니다. 밝기 조절이 가능합니다.

OLED 디스플레이

128x64 SSD1306 OLED에 메시지를 표시합니다.
