# ARES2 테스트 결과

이 폴더는 **ARES2 프로젝트의 테스트 결과를 올리는 곳**입니다.

## 목적

웹 UI · 3D 시뮬레이션 · Pico 펌웨어 등 각 런타임의 **테스트 수행 결과**(스크린샷, 로그, 측정값, 검증 리포트 등)를 이 폴더에 모아 기록·공유한다.

## 담는 내용

- 기능/회귀 테스트 결과 (통과·실패 항목)
- 하드웨어 동작 테스트 (모터·LED·센서·부저 등)
- BLE 통신·명령 프로토콜 검증 결과
- 시뮬레이터 렌더·모듈 로드 검증
- 관련 스크린샷·로그·측정 데이터

## 작성 규칙

- 테스트 1건당 파일 또는 섹션으로 정리하고, **날짜(YYYY-MM-DD)** 와 **대상(웹/펌웨어/시뮬)** 을 함께 남긴다.
- 재현 방법과 기대값·실제값을 함께 기록한다.
