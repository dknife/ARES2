# ARES 블록 코딩 가이드

## 📋 블록 목록 및 설명

### 🚗 서보 모터 (주황색 - #FF8C00)
서보 모터는 360도 회전 서보로 바퀴를 제어합니다.

| 블록 | 설명 |
|------|------|
| **서보 전진 N초** | 지정한 시간(초)만큼 앞으로 전진합니다. |
| **서보 후진 N초** | 지정한 시간(초)만큼 뒤로 후진합니다. |
| **서보 좌회전 N초** | 지정한 시간(초)만큼 왼쪽으로 회전합니다. |
| **서보 우회전 N초** | 지정한 시간(초)만큼 오른쪽으로 회전합니다. |
| **서보 계속 전진** | 정지 명령이 올 때까지 계속 전진합니다. |
| **서보 계속 후진** | 정지 명령이 올 때까지 계속 후진합니다. |
| **서보 계속 좌회전** | 정지 명령이 올 때까지 계속 좌회전합니다. |
| **서보 계속 우회전** | 정지 명령이 올 때까지 계속 우회전합니다. |
| **서보 정지** | 서보 모터를 즉시 정지합니다. |

---

### ⚡ DC 모터 (노랑색 - #FFCC00)
DC 모터는 메인 구동 모터로 더 강한 힘을 제공합니다.

| 블록 | 설명 |
|------|------|
| **DC모터 전진 N초** | DC 모터로 지정한 시간만큼 전진합니다. |
| **DC모터 후진 N초** | DC 모터로 지정한 시간만큼 후진합니다. |
| **DC모터 계속 전진** | 정지 명령이 올 때까지 DC 모터로 계속 전진합니다. |
| **DC모터 계속 후진** | 정지 명령이 올 때까지 DC 모터로 계속 후진합니다. |
| **DC모터 정지** | DC 모터를 즉시 정지합니다. |

---

### 💡 LED (빨강색 - #FF5555)
5개의 LED를 개별 또는 전체 제어합니다. LED 번호는 1~5번입니다.

| 블록 | 설명 |
|------|------|
| **LED N번 켜기 (밝기 M)** | 1~5번 중 특정 LED를 지정한 밝기(0~1)로 켭니다. 변수 사용 가능. |
| **LED N번 끄기** | 특정 LED(1~5번) 또는 전체 LED를 끕니다. |
| **LED 전체 설정 [...]** | 5개 LED 밝기를 한 번에 설정합니다. 각 값은 0(끔)~1(최대). |

---

### 🖥️ 디스플레이 (보라색 - #9966FF)
128x64 OLED 디스플레이를 제어합니다.

| 블록 | 설명 |
|------|------|
| **화면에 표시: "텍스트"** | OLED 화면에 텍스트를 표시합니다. |
| **화면 지우기** | OLED 화면을 깨끗하게 지웁니다. |

---

### 🔊 소리 (하늘색 - #00CCFF)
피에조 부저로 소리를 출력합니다.

| 블록 | 설명 |
|------|------|
| **부저 N Hz로 M초 울리기** | 지정한 주파수(Hz)와 시간(초)으로 부저를 울립니다. |

**음계 참고표:**
- 도(C4): 262Hz
- 레(D4): 294Hz
- 미(E4): 330Hz
- 파(F4): 349Hz
- 솔(G4): 392Hz
- 라(A4): 440Hz
- 시(B4): 494Hz
- 높은도(C5): 523Hz

### 📡 센서 (회청색 - #5C81A6)
다양한 센서로 환경을 감지합니다.

| 블록 | 설명 |
|------|------|
| **연결 확인** | Pico와 블루투스 연결 상태를 확인합니다. 연결되면 화면에 "CONNECTED" 표시. |
| **거리 측정 → 변수** | 초음파 센서(HC-SR04)로 전방 물체까지 거리(cm)를 측정하여 변수에 저장합니다. |
| **자기장 감지 → 변수** | 자기장 센서로 자석 감지 여부(0=없음, 1=감지)를 변수에 저장합니다. |

### 🔁 제어 (파랑 - Hue 210)
프로그램 흐름을 제어합니다.

| 블록 | 설명 |
|------|------|
| **반복 N번** | 내부 블록을 지정한 횟수만큼 반복 실행합니다. |
| **참인 동안 반복** | 조건이 참인 동안 내부 블록을 계속 반복합니다. |
| **참이 될 때까지 반복** | 조건이 거짓인 동안 내부 블록을 반복합니다. |
| **만약 ~ 이면** | 조건이 참이면 내부 블록을 실행합니다. |
| **아니면 만약** | 이전 조건이 거짓이고 이 조건이 참이면 실행합니다. (if문 설정에서 추가) |
| **아니면** | 모든 조건이 거짓일 때 실행합니다. (if문 설정에서 추가) |

### 🔢 수학/논리 (파랑 - Hue 230)
숫자 계산과 논리 연산을 수행합니다.

**숫자:**
| 블록 | 설명 |
|------|------|
| **숫자** | 숫자 값을 입력합니다. |
| **A ○ B** | 두 숫자를 사칙연산합니다. (+, -, ×, ÷) |
| **랜덤 A ~ B** | 지정한 범위 내에서 무작위 정수를 반환합니다. |

**비교:**
| 블록 | 설명 |
|------|------|
| **A = B** | 두 값이 같으면 참을 반환합니다. |
| **A ≠ B** | 두 값이 다르면 참을 반환합니다. |
| **A < B** | 첫 번째 값이 작으면 참을 반환합니다. |
| **A ≤ B** | 첫 번째 값이 작거나 같으면 참을 반환합니다. |
| **A > B** | 첫 번째 값이 크면 참을 반환합니다. |
| **A ≥ B** | 첫 번째 값이 크거나 같으면 참을 반환합니다. |

**논리:**
| 블록 | 설명 |
|------|------|
| **그리고** | 두 조건이 모두 참일 때 참을 반환합니다. |
| **또는** | 두 조건 중 하나라도 참이면 참을 반환합니다. |
| **~ 이(가) 아니다** | 조건을 반전시킵니다. (참↔거짓) |
| **참 / 거짓** | 참 또는 거짓 값을 반환합니다. |

|------|---------|
| UART TX/RX | GP0, GP1 |

| LED 1~5 | GP20, GP19, GP18, GP17, GP16 |

| 자기장 센서 | GP26 |

## 💡 사용 팁

1. **센서 결과 사용하기**: 거리 측정 / 자기장 감지 블록은 결과를 변수에 저장합니다. 이 변수를 조건문에서 사용할 수 있습니다.

2. **LED 변수 사용**: LED N번 켜기에서 N에 변수를 넣으면 동적으로 LED를 제어할 수 있습니다.

3. **무한 루프 방지**: while 반복문은 최대 100회로 제한됩니다.

4. **블루투스 연결**: 먼저 "연결 확인" 블록으로 연결 상태를 테스트하세요.
